| Boundary | Description |
|----------|-------------|
| Filesystem (DB file) -> Process | The encrypted file is the only at-rest persistence; passphrase lives in process memory only (D-19) |
| Disk attacker -> Plaintext data | An attacker with the DB file but not the passphrase must see only ciphertext |
