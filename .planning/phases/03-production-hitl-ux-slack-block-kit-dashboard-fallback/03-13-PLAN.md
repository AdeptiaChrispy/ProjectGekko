|-----------|----------|-----------|-------------|-----------------|
| T-03-13-01 | Spoofing | /approvals/poll unauthenticated access | mitigate | Poll route is on authenticated router (router = APIRouter(dependencies=[Depends(require_session)])); unauthenticated → 302 /login |
| T-03-13-02 | Information Disclosure | Proposal data in poll response | accept | Poll response is session-scoped to settings.gekko_user_id; single-operator deployment (REG-03); no cross-user data |
| T-03-13-03 | Denial of Service | Poll interval too aggressive — hammers the DB | accept | 30s interval; single-user SQLite WAL; query is a simple SELECT on proposals table with user_id + status index; negligible load |
| T-03-13-SC | Tampering | npm/pip/cargo installs | accept | No new package installs in this plan |
</threat_model>
