<assumptions>
Three load-bearing Open Questions resolved here as researcher-recommended defaults, surfaced for operator review:
- **OQ#3 (start-of-day drawdown denominator) RESOLVED: persisted per-strategy open snapshot.** A market-open scheduler job snapshots each auto strategy's start-of-day value (Σ cost_basis of held tickers + realized P&L baseline) once and stores it (as an event/row); all intraday evaluations read this STABLE denominator. Reading live equity each tick gives a moving denominator that makes the % oscillate (RESEARCH Pitfall 3 / A4). Surfaced for operator confirmation — this is the single biggest anomaly design choice.
- **OQ#5 (PENDING auto-proposal cancellation state) RESOLVED: reuse PENDING→REJECTED** + a distinguishing `anomaly_demotion` audit event (lower-risk than adding a new state — RESEARCH A3).
- Drawdown = (start_of_day_value − current_value) / start_of_day_value, Decimal-exact; current_value = Σ market_value of this strategy's tickers + realized P&L today.
</assumptions>
