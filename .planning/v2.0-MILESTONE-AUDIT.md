# Milestone v2.0 (Safety & Trust) — Audit

**Status: gaps_found** — one in-scope requirement (TRUST-04/SC-4 anomaly auto-demotion) is unsatisfied because the reflex is never armed in the app lifespan; one cross-phase gap (autonomous cadence, BLOCKER-2) is largely Phase-7-deferred-by-design.

## What is solid (verified in committed source, not SUMMARY claims)

- **Single deterministic order firewall:** every order (HITL approve AND auto-execute) reaches the broker only through `OrderGuard.place_order`; pipeline kill → paper_live → universe → hard_caps → **portfolio_caps → capital_ceiling** → qty_price → pdt → t1 → market_hours → broker; `place_order` is zero-decorator; no second order path exists.
- **Trust ladder stacks on the first-live gate:** both the auto-branch and the Slack approve handler route LIVE + `first_live_trade_confirmed_at IS NULL` → `AWAITING_2ND_CHANNEL` (never direct execute). SI-4 consistent across HITL and auto.
- **Notification routing consistent:** bypass set identical in both DM wrappers; `cost_alert` + `anomaly_demotion` bypass quiet hours; `auto_execution` FYI respects them; daily digest aggregates auto-exec + anomaly lines.
- **Audit chain linear:** migrations 0001→0007 single head; all 24 written event types present in the 0007 CHECK vocabulary; zero orphans.
